# 更新日志

所有“小魏记账簿”的版本更新都会记录在这里。

## v1.4.0 - 2026-07-10

### 新增

- AI 解析结果支持在确认前逐笔调整金额、类型、类目、备注和日期，调整过程不会提前写入账本。
- 出账分类新增“燃气费”和“电费”，快速记账与 AI 识别可直接使用这两个分类。
- 已关闭的周期账单模板支持重新启用。

### 修复

- 修复清空账本、预算和周期模板时可能只完成部分删除的问题，失败时会恢复已处理的数据。
- 修复生成周期账单后模板推进失败可能留下重复账目的问题。
- 修复 AI 设置在不同页面之间状态不同步，以及 API Key 保存失败时覆盖当前有效值的问题。
- 修复 AI 返回内容包含说明文字、括号或字符串内括号时可能截取错误 JSON 的问题。
- 修复本月趋势切换为入账后，指标仍显示出账统计的问题。
- 修复当前月平均值计入未来日期、预算小数被取整和零进度仍显示色条的问题。
- 修复 CSV 中以 `=`、`+`、`-`、`@` 开头的备注或类目可能被表格软件当作公式执行的问题。
- 修复旧提示计时器可能提前关闭新提示的问题。

### 优化

- 记录图片改为缩略图解码并增加缓存，同时限制图片文件名，降低大图内存占用和异常路径风险。
- 分享面板无法打开、清空 API Key 失败时会显示更准确的提示。
- 金额输入统一保留有效小数，并拒绝 `NaN`、`Infinity` 等无效数值。
- 同步 App 内版本号为 1.4.0（Build 6）。

### 说明

- 本版本生成的 ipa 为 unsigned，需要用户自行使用自签工具签名后安装。
- AI 功能需要用户自行配置 OpenAI 兼容接口和 API Key，正式 Release 不内置 API Key。

## v1.3.1 - 2026-07-07

### 修复

- 修复 JSON 文件损坏时可能被空数据覆盖的问题，损坏文件会备份为 `.corrupt-时间戳`。
- 修复“清空全部数据”未清理预算、周期模板、AI 设置和 API Key 的问题。
- 修复 AI Base URL 允许明文 HTTP 的安全问题，现在仅允许 HTTPS。
- 修复 AI 请求错误展示上游 body 的问题，避免泄露敏感信息。
- 修复周期账单多期逾期时可能被压成一笔的问题。
- 修复多笔 AI 保存不是事务式的问题，失败时会回滚内存状态。
- 同步 App 内版本号为 1.3.1。

### 优化

- AI 示例改为更通用的日常记账文本，移除店名强相关示例。
- “清空全部数据”增加精确文本确认，降低误删风险。

### 说明

- 本版本生成的 ipa 为 unsigned，需要用户自行使用自签工具签名后安装。
- AI 功能需要用户自行配置 OpenAI 兼容接口和 API Key，正式 Release 不内置 API Key。

## v1.3.0 - 2026-07-07

### 新增

- AI 记账助手支持一句话解析多笔账单，例如同时识别工资入账和外卖出账。
- AI 解析结果确认卡片支持展示多笔记录，并可一次确认保存全部账单。
- 新增多笔 AI 账单保存逻辑，保存后会逐笔写入本地账本并更新统计金额。
- 新增多笔 AI 解析与批量保存测试，覆盖 2 笔和 25 笔账单场景。

### 优化

- AI 接口提示词明确要求多笔账单按发生顺序返回，不合并金额、不丢弃记录、不限制记录数量。
- AI 响应解析支持顶层数组，以及 `records`、`items`、`transactions`、`entries`、`results` 等多种结果包裹格式。
- DEBUG Mock 解析增强，可在本地验证多笔账单识别和保存流程。
- AI 确认页在多笔账单时显示“第几笔”、类型标签、单笔调整入口和“确认记账 N 笔”。

### 说明

- 本版本生成的 ipa 为 unsigned，需要用户自行使用自签工具签名后安装。
- AI 功能需要用户自行配置 OpenAI 兼容接口和 API Key，正式 Release 不内置 API Key。

## v1.2.0 - 2026-07-07

### 新增

- 新增 AI 记账助手，可用自然语言描述账单并解析为金额、类型、类目、备注和日期。
- 新增 AI 解析结果确认卡片，确认后才会写入本地账本。
- 新增 AI 设置模块，支持配置 API Base URL、备用 Base URL、API Key 和模型名称。
- 支持从 OpenAI 兼容接口拉取模型列表，也可以手动填写模型名称。
- DEBUG 构建新增本地 Mock 解析开关，用于没有真实 API Key 时验证界面流程。

### 优化

- API Key 优先保存到 iOS Keychain，不写入账本 JSON、导出 JSON、README、测试数据或截图。
- AI 解析会在本地二次校验金额、类型、类目、备注长度和日期，避免模型结果直接静默写入账本。
- 首页新增“AI 记一笔”入口，并保留确认后保存 / 手动调整 / 取消流程。
- 快速记账预设调整为燃气费、电费、房租等常用固定支出。
- 移除独立双花瓣装饰资源和相关 UI 文案，减少公开仓库中的授权风险。

### 说明

- 本版本生成的 ipa 为 unsigned，需要用户自行使用自签工具签名后安装。
- AI 功能需要用户自行配置 OpenAI 兼容接口和 API Key，正式 Release 不内置 API Key。

## v1.1.0 - 2026-07-05

### 新增

- 支持为每笔账选择一张记录图片。
- 新增记录图片本地存储目录：`Documents/BeanLedgerRecordImages/`。
- 首页最近记录支持显示用户上传图片缩略图。
- 未上传图片时，按账目类型和二级类目自动显示可爱风格类目小图。

### 优化

- 保存账目图片时自动压缩为较小 JPEG 文件，避免把大图写入账本 JSON。
- 删除单笔账或清空账本时，同步清理对应的本地图片文件。
- README 补充记录图片的数据存储和使用说明。

### 说明

- 本版本生成的 ipa 为 unsigned，需要用户自行使用自签工具签名后安装。
