# 更新日志

所有“小魏记账簿”的版本更新都会记录在这里。

## v1.2.0 - 2026-07-07

### 新增

- 新增 AI 记账助手，可用自然语言描述账单并解析为金额、类型、类目、备注和日期。
- 新增 AI 解析结果确认卡片，确认后才会写入本地账本。
- 新增 AI 设置模块，支持配置 API Base URL、备用 Base URL、API Key 和模型名称。
- 支持从 OpenAI 兼容接口拉取模型列表，也可以手动填写模型名称。
- DEBUG 构建新增本地 Mock 解析开关，用于没有真实 API Key 时验证界面流程。

### 优化

- API Key 优先保存到 iOS Keychain，不写入账本 JSON、导出 JSON、README、测试数据或截图。
- AI 解析会在本地二次校验金额、类型、类目、备注长度和日期，避免模型结果直接静默写入账本。
- 首页新增“AI 记一笔”入口，并保留确认后保存 / 手动调整 / 取消流程。
- 快速记账预设调整为燃气费、电费、房租等常用固定支出。
- 移除独立双花瓣装饰资源和相关 UI 文案，减少公开仓库中的授权风险。

### 说明

- 本版本生成的 ipa 为 unsigned，需要用户自行使用自签工具签名后安装。
- AI 功能需要用户自行配置 OpenAI 兼容接口和 API Key，正式 Release 不内置 API Key。

## v1.1.0 - 2026-07-05

### 新增

- 支持为每笔账选择一张记录图片。
- 新增记录图片本地存储目录：`Documents/BeanLedgerRecordImages/`。
- 首页最近记录支持显示用户上传图片缩略图。
- 未上传图片时，按账目类型和二级类目自动显示可爱风格类目小图。

### 优化

- 保存账目图片时自动压缩为较小 JPEG 文件，避免把大图写入账本 JSON。
- 删除单笔账或清空账本时，同步清理对应的本地图片文件。
- README 补充记录图片的数据存储和使用说明。

### 说明

- 本版本生成的 ipa 为 unsigned，需要用户自行使用自签工具签名后安装。

## v1.0.0 - 2026-07-05

### 新增

- 完成基础记账功能：出账、入账、攒豆豆、借贷。
- 支持 JSON 本地持久化。
- 支持快速记账。
- 支持历史备注提示。
- 支持首页统计、全部记录筛选、统计页。
- 支持本月趋势图。
- 支持预算功能。
- 支持周期账单 / 定时记账模板。
- 支持收支日历。
- 支持账单搜索增强。
- 支持 JSON / CSV 导出。
- 支持 URL Scheme：`beanledger://add`。
- 完成 iOS AppIcon 和 App 显示名称“小魏记账簿”。

### 优化

- 统一收支类型语义色：
  - 出账：黄色
  - 入账：蓝色
  - 攒豆豆：粉紫色
  - 借贷：紫色
- 优化 iOS AppIcon 安全边距。
- 优化金额正负号显示。
- 优化统计页可读性。

### 说明

- 本版本生成的 ipa 如为 unsigned，需要用户自行使用自签工具签名后安装。
