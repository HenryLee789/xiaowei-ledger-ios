# 更新日志

所有“小魏记账簿”的版本更新都会记录在这里。

## v1.0.0 - 2026-07-05

### 新增

- 完成基础记账功能：出账、入账、攒豆豆、借贷。
- 支持 JSON 本地持久化。
- 支持快速记账。
- 支持历史备注提示。
- 支持首页统计、全部记录筛选、统计页。
- 支持本月趋势图。
- 支持预算功能。
- 支持周期账单 / 定时记账模板。
- 支持收支日历。
- 支持账单搜索增强。
- 支持 JSON / CSV 导出。
- 支持 URL Scheme：`beanledger://add`。
- 完成 iOS AppIcon 和 App 显示名称“小魏记账簿”。

### 优化

- 统一收支类型语义色：
  - 出账：黄色
  - 入账：蓝色
  - 攒豆豆：粉紫色
  - 借贷：紫色
- 优化 iOS AppIcon 安全边距。
- 优化金额正负号显示。
- 优化统计页可读性。

### 说明

- 本版本生成的 ipa 如为 unsigned，需要用户自行使用自签工具签名后安装。
